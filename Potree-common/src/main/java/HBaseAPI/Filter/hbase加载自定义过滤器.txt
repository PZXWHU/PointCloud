将过滤器打包成jar，放到服务器上的hbase 的classpath下面，即hbase/lib。
必须放在服务器的每一个主机上面，否则会出现找不到class 的异常
HBASE_CLASSPATH=<path to your custom jar on local file system on all region servers>


还有一种方法，是将jar放在hdfs上面，下面这段配置放到所有hbase的配置文件中
<property>
  <name>hbase.dynamic.jars.dir</name>动态加载，即可在不关闭hbase的情况下加载jar包
  <value>hdfs://myhdfs.host:8020/hbase/lib</value>默认的是本地的hbase/lib，这样的唯一好处就是不用复制jar包到各个机器上
  <final>true</final>
</property>
